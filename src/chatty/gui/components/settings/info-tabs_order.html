An order can be specified for:

  • A specific tab (name of the channel, all-lowercase), takes precedence over type if available
  • A type of tab (# - Channel, $ - Whisper, - - Info Panel/Docked Dialog, these are mirrored by the dedicated tab order settings)

Lower numbers mean the tab is earlier in the order, so -3 would be sorted before -2 and 1. A value of 0 is the default, so negative values put tabs in front and positive values after tabs that are not listed here (neither by type or specific name).

These can also be changed in the tab context menu.
